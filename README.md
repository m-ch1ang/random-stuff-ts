# Heng Gin Sales App

A modern full-stack web application built with **React + TypeScript** and **Supabase**.  
Designed for managing BOMs and creating purchase orders.

## 🧱 Tech Stack Overview
### Frontend
- ⚛️ **React**
- 🔠 **TypeScript**

### Backend
- 🧩 **Supabase**
